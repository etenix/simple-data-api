# Simple Data API

This project is a simple REST API built with Python and Flask.

It provides basic data processing functions such as sum and statistics calculation.

## Features

- REST API using Flask
- JSON input/output
- Calculate sum of numbers
- Calculate statistics (max, min, average)

## Tech Stack

- Python
- Flask
- REST API
